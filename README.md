# 📦 Smart Inventory Analytics Dashboard

An interactive dashboard to analyze inventory data of 5,000+ products.

## 🚀 Features
- Dynamic filtering (category, stock status, search)
- KPI metrics (stock, turnover, alerts)
- Interactive charts (bar, pie, line, top products)
- Inventory insights for decision-making

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js
- Canvas.js

## 📊 Insights
- Identify low/critical stock items
- Track product performance
- Analyze category distribution
- Monitor sales trends
